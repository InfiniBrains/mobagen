# MoBaGEn
Welcome to the **MoBaGEn**. The **MO**dule **BA**sed **G**ame **EN**gine. 

**MoBaGEn** is a free lightweight, cross-platform 3D Game Engine implemented in C++ and scripted in C#(Mono), Javascript(V8) and Lua.
